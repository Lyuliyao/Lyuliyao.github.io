Vector Function Trace

            
int count_positive(const vector<int>& v) {
    int count = 0;
    for (size_t i = 0; i < v.size(); ++i) {
        if (v[i] > 0) {
            ++count;
        }
    }
    return count;
}
  • Identify whether the vector is copied.
  • Trace the counter only when the condition is true.
  • Check bounds before indexing.

Review Drill: Vector and Class Function

            
class Student {
public:
    Student(string n, double s) : name(n), score(s) {}
    double get_score() const { return score; }

private:
    string name;
    double score;
};

double average_score(const vector<Student>& roster) {
    double total = 0;
    for (size_t i = 0; i < roster.size(); ++i) {
        total += roster[i].get_score();
    }
    return total / roster.size();
}
  • const vector<Student>& avoids copying the roster.
  • get_score() must be const because the roster is read-only.
  • Before using this in real code, decide what happens for an empty roster.

Review Drill: Const-Correct Interface

            
class BankAccount {
public:
    BankAccount(double initial);
    void deposit(double amount);
    double get_balance() const;
};

const BankAccount acct(100);
cout << acct.get_balance() << endl;
// acct.deposit(5);  // error: deposit may mutate acct
  • A const object can call only const member functions.
  • Accessor functions usually should be marked const.
  • Mutating functions should not pretend to be read-only.

Lecture 25 Code Review

Trace it

  • Q1. What should you trace first? 
                
int max_value(const vector<int>& v) {
    int best = v[0];
    for (size_t i = 1; i < v.size(); ++i) {
        if (v[i] > best) {
            best = v[i];
        }
    }
    return best;
}
  • Answer: Trace best and start the loop at index 1 because index 0 initializes the answer.
