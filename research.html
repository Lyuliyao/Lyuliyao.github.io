Machine Learning and Multiscale Modeling

Developing innovative machine learning methodologies to enhance bottom-up multiscale modeling and accurately predict complex physical phenomena.

  • A non-conservative memory term is critical for coarse-grained molecular dynamics (CGMD).
    1. Many-body Dissipative Interactions:
      Introduced a machine-learning-based coarse-grained molecular dynamics (CGMD) model leveraging the Mori-Zwanzig formalism to include non-Markovian many-body memory interactions.
    2. Neural Networks Respect Physical Symmetries:
      Design neural networks that respect physical symmetries (e.g., rotational, translational, permutation) to model complex friction matrix accurately.
    3. Accuracy beyond Mean-field Metrics:
      Instead of matching the mean-field metrics such as the velocity autocorrelation function, the present model is constructed based on the Mori-Zwanzig formalism and naturally inherits the heterogeneous information.
  • Coarse-Grained Molecular Dynamics Models for Non-Equilibrium Processes
    1. Generalization Challenge:
      Applicability to various non-equilibrium processes is generally unwarranted due to the inconsistency in the conditional distribution.
    2. Training CG variables:
      Seeking a set of auxiliary CG variables based on the time-lagged independent component analysis to minimize the entropy contribution of the unresolved variables.

Algorithms of Scientific Machine Learning

  • Deep Mixed Residual Method (MIM) design for solving high-order partial differential equations
    1. Mixed Residual:
      Rewrite a high-order PDE into a first-order system,in the same spirit as local discontinuous Galerkin method and mixed finite element method
    2. Flexibility in Network Design:
      Allows for different configurations, such as using the same or different DNNs for various unknown functions.
    3. Enforce Boundary Condition :
      Design DNNs satisfy boundary and initial conditions automatically in a systematic manner
    • Deep Learning Based Discontinuous Galerkin Method (D2GM) to solve hyperbolic equations with discontinuous solutions and random uncertainties
    1. Discontinuities of the solutions:
      Using the weak formulation and discontinuous element basis, the D2GM is able to approximate discontinuous PDE solutions with high accuracy
    2. High-dimensional Random Space:
      Using the mini-batch sampling with controllable number of samples, the D2GM overcomes the curse of dimensionality.
