Array Parameter

            
void print(const int* a, int size) {
    for (int i = 0; i < size; ++i) {
        cout << a[i] << " ";
    }
}
  • Array parameters usually need a separate size.
  • The function receives a pointer to the first element.
  • The size is not automatically known.

Decay Trap

            
void f(int a[]) {
    cout << sizeof(a) << endl;
}

int data[5];
cout << sizeof(data) << endl;
f(data);
  • In the caller, data is an array.
  • In the function, a is a pointer.
  • Do not use sizeof on a parameter to find array length.

C Strings

            
char word[] = "hi";
cout << word << endl;
cout << word[0] << endl;
  • A C string is a char array ending with \0.
  • String literals provide null-terminated character data.
  • Prefer std::string in normal PIC 10A code.

Worked Example: Sum Array Parameter

            
int sum(const int* a, int size) {
    int total = 0;
    for (int i = 0; i < size; ++i) {
        total += a[i];
    }
    return total;
}

int values[4] = {2, 4, 6, 8};
cout << sum(values, 4) << endl;
  • The pointer tells the function where the first element is.
  • The size tells the function when to stop.
  • const int* says the function will not change the elements.

Worked Example: Walk a C String

            
char word[] = "cat";
int i = 0;

while (word[i] != '\0') {
    cout << word[i] << endl;
    ++i;
}
  • The loop stops at the null terminator.
  • The terminator is stored in the array but is not printed as a normal letter.
  • For everyday string work, std::string is safer and easier.
