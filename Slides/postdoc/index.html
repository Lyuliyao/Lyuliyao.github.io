Machine learning methodology for multiscale modeling

Following the Mori-Zwanzig formalism, the dynamics of CG variable \(\bQ, \bP\) takes \[ \dot{\mathbf{P}} = \underbrace{-\nabla U(\mathbf{Q})}_{\text{conservative part}} - \underbrace{\int_0^t \color{red}{\mathbf{K}(\mathbf{Q}(s), t-s) }\dot \bQ(s) ds}_{\text{Memory part}} + \underbrace{\mathbf{R}(t)}_{\text{Random part}} \]
  1. Many-body Dissipative Interactions
    Markovian-Pairwise: \( \dot{\mb P_i} = \mb F_i^C - \sum_{j} \mb K(Q_{ij})\mb V_j + \mb R(t)\)
    Non Markovian-Isotropic: \( \dot{\mb P} = \mb F^C - \int_0^t \mb K(t-s) \mb V(s) \diff s + \mb R(t)\)
    (Our Model) Non Markovian-Manybody: \(\dot{\mb P} = \mb F^C - \bm\Xi(\mb Q)\int_0^t {\rm e}^{- \bm\Lambda (t-s)}\bm\Xi(\mb Q)^T \mb V_{ij}(s) \diff s + \mb R(t).\)
  2. Neural Networks Respect Physical Symmetries
    Translation: \(\mathbf{\Xi}(\bQ_1 +\mathbf{b},\cdots,\bQ_M +\mathbf{b}) = \mathbf{\Xi}(\bQ_1,\cdots,\bQ_M)\)
    Rotation: \(\mathbf{\Xi}_{ij}(\mathrm{U}\bQ_1,\cdots,\mathrm{U}\bQ_M) = \mathrm{U}\mathbf{\Xi}_{ij}(\bQ_1,\cdots,\bQ_M)\mathrm{U}^T\)
    Permutation Symmetries: \(\mathbf{\Xi}_{ij}(\bQ_{\sigma(1)},\cdots,\bQ_{\sigma(M)}) = \mathbf{\Xi}_{\sigma(i)\sigma(j)}(\bQ_1,\cdots,\bQ_M)\)

Consensus-Based Optimization and Sampling

Let \(J:\mathbb R^d \to \mathbb R\) be a function. The problem we consider is given by \(\arg\min_{\Theta\in \mathbb R ^d} f(\Theta)\).
Original Consensus-Based Optimization
\( \Theta^i_{t+1} = \Theta^i_t -\lambda (\Theta^i_t - M)+\sqrt{\lambda}\sigma|(\Theta^i_t - M)|\xi_{\theta}^i, \) where \(M=\sum\limits_{k} \frac{\Theta_t^k w^k}{\sum\limits_{i} w^i}\), \(w^i = \exp\left ( -\beta \mathcal J^i\right ) \) and \(\xi_{\theta}^i\sim \mathcal N(0,1)\).
Momentum based Consensus-Based Optimization
\[ \begin{aligned} \Theta^i_{t+1} = \Theta^i_t +\lambda \Omega^i_t - \gamma_1\lambda (\Theta^i_t - M)+\sqrt{\lambda}\xi_{\theta}^i, \quad \Omega^i_{t+1} = \Omega^i_t - \lambda m (\Theta^i_t - M) -\lambda \gamma_2 \Omega^i_t + \sigma(t)\sqrt{\lambda m} \xi_{\omega}^i. \end{aligned} \]
The mean-field limit of the model is given by
\begin{equation} \label{equ:mean_M_CBO} \begin{aligned} \intd \bar{\Theta}_t = \bar\Omega_t \intd t -\gamma_1 \l (\bar\Theta_t - \mathcal{M}_\beta[\mu_t] \r )\intd t+\sigma(t)\intd W_{\theta,t},\quad \intd \bar{\Omega}_t = - m \l (\bar\Theta_t - \mathcal{M}_\beta[\mu_t] \r ) \intd t -\gamma_2 \bar \Omega_t \intd t + \sqrt{m} \sigma(t)\intd W_{\omega,t}, \end{aligned} \end{equation} where \( \CM[\mu] = \frac{\int \theta \exp(-\beta \mathcal J(\theta)\mu(\intd \theta)}{\int \exp(-\beta \mathcal J(\theta)\mu(\intd \theta)} \), \(\mu_t(\theta) = \int \rho_t(\theta,\intd \omega)\), and \(\rho_t = Law (\bar{\Theta}_t,\bar\Omega_t)\).
Then the corresponding Fokker-Planck equation is
\begin{equation} \label{equ:FP_M_CBO} \begin{aligned} \partial_t \rho_t =& -\nabla_\theta\cdot \l ( \l ( \omega-\gamma_1(\theta-\CM [\mu_t])\r ) \rho_t\r ) + \nabla_\omega\cdot \l ( \l( m \l (\theta - \CM \l [\mu_t \r] \r ) +\gamma_2 \omega\r)\rho_t\r ) +\frac{\sigma(t)^2m}{2} \Delta_\omega \rho_t+\frac{\sigma(t)^2 }{2} \Delta_\theta \rho_t. \end{aligned} \end{equation}
Define the energy functional as \( E[\rho] = \frac{1}{2} \int \|\theta-\tilde\theta\|^2+ m^{-1}\|\omega\|^2 \intd\rho, \) we have the exponential decay \(E[\rho_t] \leq E[\rho_0]\exp(-(1-\tau)\lambda t)\) and, up to a constant, the same behavior for \(W^2_2(\rho_t,\delta_{(\tilde\theta,0)})\).
Application:
  • Optimization: high dimensional (up to \(1000\)) nonconvex target function, Deep Ritz for solivng PDE.
  • Enhance sampling for construction free energy surface.
  • Stochastic (mean field) optimal control: update the policy without taking the gradient.

Algorithms of Scientific Machine Learning

  • Deep Mixed Residual Method (MIM) design for solving high-order partial differential equations
    1. Mixed Residual:
      Rewrite a high-order PDE into a first-order system,in the same spirit as local discontinuous Galerkin method and mixed finite element method
    2. Flexibility in Network Design:
      Allows for different configurations, such as using the same or different DNNs for various unknown functions.
    3. Enforce Boundary Condition :
      Design DNNs satisfy boundary and initial conditions automatically in a systematic manner
    • Deep Learning Based Discontinuous Galerkin Method (D2GM) to solve hyperbolic equations with discontinuous solutions and random uncertainties
    1. Discontinuities of the solutions:
      Using the weak formulation and discontinuous element basis, the D2GM is able to approximate discontinuous PDE solutions with high accuracy
    2. High-dimensional Random Space:
      Using the mini-batch sampling with controllable number of samples, the D2GM overcomes the curse of dimensionality.
