Declaring Vectors

            
#include <vector>
#include <string>

vector<int> scores;
vector<double> weights;
vector<string> names;
  • Include <vector>.
  • The type inside angle brackets is the element type.
  • All elements in one vector have the same type.

Growing and Shrinking

            
vector<int> v;
v.push_back(10);
v.push_back(20);
cout << v.size() << endl;
v.pop_back();
  • push_back appends one element.
  • pop_back removes the last element.
  • size() reports the current number of elements.

Indexing

            
vector<int> v = {4, 7, 9};
cout << v[1] << endl;
cout << v.at(2) << endl;
  • Indices start at zero.
  • [] out of range is undefined behavior.
  • at() out of range is a runtime error.

Design Questions

Before writing code

  • What data changes over time?
  • What function owns each task?
  • What type communicates the intent?

Before debugging

  • What should happen on a tiny input?
  • Where can scope or lifetime end?
  • Which operation may access invalid memory?

Vector Constructors

Size and fill value

              
vector<int> zeros(5);
vector<int> eights(5, 8);

cout << zeros.size() << endl;
cout << eights[0] << endl;

Initializer list

              
vector<string> names = {"Ada", "Grace"};

names.push_back("Katherine");
cout << names.size() << endl;
A vector has a current size. It can start empty, start with repeated values, or start from listed values.

More Vector Member Functions

            
vector<int> v = {1, 2, 3};

cout << v.empty() << endl;

v.clear();
cout << v.size() << endl;

v.resize(4);
cout << v.size() << endl;
  • empty() asks whether the size is zero.
  • clear() removes all elements.
  • resize(n) changes the current size and value-initializes new elements.
