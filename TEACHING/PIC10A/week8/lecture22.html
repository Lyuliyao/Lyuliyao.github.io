Default Constructor

            
struct Timer {
    int seconds;

    Timer() {
        seconds = 0;
    }
};
  • A constructor runs when an object is created.
  • A default constructor takes no arguments.
  • It should establish a valid starting state.

Parameterized Constructor

            
struct Point {
    double x;
    double y;

    Point(double x_in, double y_in) {
        x = x_in;
        y = y_in;
    }
};
  • Constructor parameters initialize member variables.
  • The call provides the starting data.
  • Use meaningful parameter names.

Constructor Calls

            
Point origin(0, 0);
Point p(3, 4);
Counter c;
  • Objects are constructed before they are used.
  • Different constructors support different creation patterns.
  • The declaration determines which constructor is called.

Design Questions

Before debugging

  • Which operation may access invalid memory?

Constructor Trace

            
struct Rectangle {
    double width;
    double height;

    Rectangle(double w, double h) {
        width = w;
        height = h;
    }
};

Rectangle r(3, 4);
  • Object storage is created first.
  • Constructor parameters w and h receive the arguments.
  • The constructor body assigns member variables into a valid state.

Default Constructor Motivation

            
struct Student {
    string name;
    int score;

    Student() {
        name = "unknown";
        score = 0;
    }
};

Student s;
  • A default constructor makes no-argument construction meaningful.
  • It prevents objects from starting in an unclear state.
  • Every constructor should establish class invariants.
