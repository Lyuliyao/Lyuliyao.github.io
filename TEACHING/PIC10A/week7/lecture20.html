Overloading and Default Arguments

C++ can choose among functions with the same name when parameter lists differ. Default arguments add another way to simplify calls.

Legal Overloading

            
int add(int a, int b);
double add(double a, double b);
  • Same name is allowed if parameter lists differ.
  • The compiler chooses the best matching signature.
  • This is function overloading.

Not Distinguished by Return

            
int f(int x);
double f(int x);   // error
  • Return type alone cannot distinguish overloads.
  • The call f(3) would be ambiguous.
  • Change the parameter list or the name.

Default Arguments

            
void greet(string name = "friend") {
    cout << "Hello, " << name << endl;
}

greet("Ada");
greet();
  • A default argument is used when the caller omits an argument.
  • Defaults should appear in the declaration.
  • Parameters after a default usually need defaults too.

Design Questions

Before writing code

  • What data changes over time?
  • What function owns each task?
  • What type communicates the intent?

Before debugging

  • What should happen on a tiny input?
  • Where can scope or lifetime end?
  • Which operation may access invalid memory?

Overload Resolution Examples

            
void print(int n) {
    cout << "int: " << n << endl;
}

void print(double x) {
    cout << "double: " << x << endl;
}

print(3);
print(3.5);
  • The compiler chooses the overload with the best parameter match.
  • print(3) matches int exactly.
  • print(3.5) matches double exactly.

Default Argument Rules

            
void print(const vector<int>& v, size_t up_to = 0);

// Better meaning:
// up_to == 0 means print the whole vector.

void print(const vector<int>& v, size_t up_to) {
    if (up_to == 0 || up_to > v.size()) {
        up_to = v.size();
    }
    for (size_t i = 0; i < up_to; ++i) {
        cout << v[i] << " ";
    }
}
  • Default arguments let a call omit less-common options.
  • The default belongs with the declaration students see first.
  • Use defaults only when the omitted behavior is obvious.
