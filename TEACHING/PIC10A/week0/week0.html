Doing "nothing"

 
                        int main() {
                            return 0;
                        }
  • Every C++ program must have exactly one main function.
  • Execution begins with main.
  • Semicolons (;) mark the end of statements.
  • return 0 means: “Program finished successfully.”

Hello World!

 
                        #include <iostream>
                        int main() {
                            std::cout << "Hello, World!\n";
                            return 0;
                        }
  • #include <iostream> is a header file library that lets us work with input and output objects, such as "std::cout".
  • std::cout is the standard output stream object in C++.
  • The << operator is used to send data to the output stream.
  • "Hello, World!\n" is a string literal that will be printed to the console, followed by a newline character (\n).

Hello World! Again 

 
                        #include <iostream>
                        using namespace std;
                        int main() {
                            cout << "Hello, World!" << endl;
                            return 0;
                        }
  • using namespace std; allows us to use names from the standard library without the std:: prefix.
  • endl is an output manipulator that inserts a newline character and flushes the output buffer.

Backslash Escape Sequences

  • \n - Newline
  • \t - Tab
  • \\ - Backslash
  • \" - Double Quote

#include <iostream>
using namespace std;
int main() {
    cout << "Hello, \nWorld!\n";
    cout << "This is a tab:\tSee?\n";
    cout << "She said, \"Hello!\"\n";
    cout << "This is a backslash: \\\n";
    return 0;
}
