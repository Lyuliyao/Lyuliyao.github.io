A Simple Struct

            
struct Book {
    string title;
    int pages;
};

Book b;
b.title = "C++";
b.pages = 300;
  • A struct groups related values.
  • Each instance has its own member variables.
  • Use the dot operator to access members.

Member Function

            
struct Point {
    double x;
    double y;

    double radius() const {
        return sqrt(x * x + y * y);
    }
};
  • A member function belongs to the struct/class.
  • It can use member variables directly.
  • The call looks like p.radius().

Free vs Member

            
double radius(Point p);

Point p;
radius(p);
p.radius();
  • A free function receives the object as an explicit argument.
  • A member function is called through an object.
  • Choose the style that makes the interface clear.

Design Questions

Before writing code

  • What data changes over time?
  • What function owns each task?
  • What type communicates the intent?

Before debugging

  • What should happen on a tiny input?
  • Where can scope or lifetime end?
  • Which operation may access invalid memory?

Rectangle Example

            
struct Rectangle {
    double width;
    double height;
};

double area(Rectangle r) {
    return r.width * r.height;
}

Rectangle box;
box.width = 3;
box.height = 4;
cout << area(box) << endl;
  • A struct groups related member variables into one type.
  • The free function receives a rectangle as an explicit argument.
  • This is the first step toward objects with behavior.

Move Behavior Into the Type

            
struct Rectangle {
    double width;
    double height;

    double area() const {
        return width * height;
    }
};

Rectangle box{3, 4};
cout << box.area() << endl;
  • A member function is called through an object.
  • Inside the member function, member variables can be used directly.
  • The interface now reads like “ask this rectangle for its area.”
