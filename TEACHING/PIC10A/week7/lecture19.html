Header and CPP Organization

Large programs split declarations and definitions across files. This lecture explains what belongs in headers and what belongs in cpp files.

Roadmap

Header File

            
// temperature.hpp
#ifndef TEMPERATURE_HPP
#define TEMPERATURE_HPP

double celsius_to_fahrenheit(double c);

#endif
  • Headers contain declarations.
  • Function comments usually belong with declarations.
  • Include guards prevent accidental repeated inclusion.

CPP File

            
// math_tools.cpp
#include "math_tools.hpp"

int square(int n) {
    return n * n;
}
  • CPP files contain definitions.
  • They include their own header.
  • The compiler checks that declarations and definitions agree.

Main File

            
// main.cpp
#include "math_tools.hpp"

int main() {
    cout << square(5) << endl;
}
  • Main includes the header, not the cpp file.
  • The project build compiles cpp files separately.
  • The linker connects calls to definitions.

Design Questions

Before writing code

  • What data changes over time?
  • What function owns each task?
  • What type communicates the intent?

Before debugging

  • What should happen on a tiny input?
  • Where can scope or lifetime end?
  • Which operation may access invalid memory?

File Layout Example

functions.hpp

              
#ifndef FUNCTIONS_HPP
#define FUNCTIONS_HPP

#include <vector>

int sum(const std::vector<int>& v);
void print(const std::vector<int>& v);

#endif

functions.cpp

              
#include "functions.hpp"
#include <iostream>

using namespace std;

int sum(const vector<int>& v) {
    int total = 0;
    for (size_t i = 0; i < v.size(); ++i) {
        total += v[i];
    }
    return total;
}

How the Build Connects Files

            
// main.cpp
#include "functions.hpp"
#include <vector>

using namespace std;

int main() {
    vector<int> v = {1, 2, 3};
    print(v);
    return sum(v);
}
  • main.cpp sees declarations through the header.
  • functions.cpp provides definitions.
  • The linker connects the compiled call to the compiled definition.
